# Take-Some-Notes
    This project was created with a user in mind, that needs to be able to track important info or tasks. This application will allow the user to create notes, recall them and delete them.

### Technologies Used:
- *Node.js*
- *JSON*
- *Express*
- *CSS*
- *HTML*
- *Javascript*
- *Heroku*
